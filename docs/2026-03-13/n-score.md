# Beyond Binary Success: Sample-Efficient and Statistically Rigorous Robot Policy Comparison

**日期**: 2026-03-13  
**arXiv**: [2603.13616](https://arxiv.org/abs/2603.13616)  
**代码**: 未公开  
**领域**: 机器人 / 评估方法论  
**关键词**: policy comparison, sequential testing, SAVI, sample efficiency, progress metrics

## 一句话总结
提出 N-SCORE，一个基于安全随时有效推断（SAVI）的机器人策略对比框架——支持从二值成功率到连续奖励的通用评估指标，通过序贯检验在统计严格性不降的前提下，比批处理方法减少 70% 评估负担，比二值序贯方法减少 50%。

## 研究背景与动机
- 通才机器人策略越来越强，但评估限于少量硬件 rollout（通常 10-60 次）
- 二值成功率是事实标准——但完成 90% 任务和完全不动都是"失败"，丢失信息
- 现有方法在统计严格性、样本效率、指标通用性三者间只能满足部分

## 方法详解

### N-SCORE 框架
1. **基于 SAVI 的序贯检验**: 证据足够时随时可停——不需要预设样本量
2. **非参数方法**: 适用于任意有界进展指标（二值、离散部分学分、连续奖励等）
3. **核密度估计 + 比较特化**: 低维设定下快速适应数据结构
4. **Type-1 Error 严格控制**: 保证错误判断"新策略更好"的概率 ≤ α*

## 实验关键数据
- 4500+ 硬件 rollout + 2000 高保真仿真 rollout 验证
- 比批处理方法减少 70% 评估量
- 比 STEP（二值 SOTA）减少 50% 评估量
- 细粒度进展指标比二值成功率能更快区分策略差异

## 亮点与洞察
- **超越二值成功率**的呼吁非常及时——机器人社区长期依赖粗糙指标
- 序贯检验的"随时可停"特性完美匹配硬件评估的资源约束

## 评分
- 新颖性: ⭐⭐⭐⭐ 首次将 SAVI 框架系统化应用于通用机器人策略比较
- 实验充分度: ⭐⭐⭐⭐⭐ 大规模硬件+仿真数据验证，多种指标覆盖
- 价值: ⭐⭐⭐⭐⭐ 对机器人评估方法论有范式性影响
