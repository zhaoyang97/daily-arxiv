# Cross-modal Learning for Plankton Recognition

**日期**: 2026-03-17  
**arXiv**: [2603.16427](https://arxiv.org/abs/2603.16427)  
**领域**: 多模态/VLM  
**关键词**: 浮游生物识别, CLIP对比学习, 跨模态, 少样本, 光学剖面

## 一句话总结
将 CLIP 式对比学习应用于显微图像+光学剖面（散射/荧光信号）的跨模态浮游生物识别，域内 96% 准确率，跨域 90%+，仅光学信号也达 60%。

## 研究背景与动机

1. **领域现状**: 自动化浮游生物分类对海洋生态监测至关重要。CytoSense 流式细胞仪同时产生明场图像+光学剖面。

2. **现有痛点**: 人工标注昂贵；光学剖面数据未被利用；跨采样站域偏移降低泛化性。

3. **核心 idea**: 用对比学习将图像和光学剖面对齐到共享空间——图像提供丰富视觉，光学提供物理测量。k-NN 分类避免重新训练。

## 方法详解

### 整体框架
1. **双编码器**: ViT-Tiny 编码显微图像 / 1D CNN 编码光学剖面
2. **对比预训练**: InfoNCE + Sigmoid 换失对齐配对样本
3. **k-NN 分类**: 在共享空间中用少量标注样本做最近邻
4. **三种推理模式**: 图像+剖面 / 仅图像 / 仅剖面

## 实验关键数据

| 设置 | 准确率 |
|------|--------|
| 域内 图像+剖面 | **96.01%** |
| 域内 仅图像 DINO | ~93% |
| 跨域 (LAB→UTO) | **90%+** |
| 仅光学剖面 | 60% |

- 对比学习后图像特征 > 纯 DINO——光学提供了补充监督
- 弱模态（60%）也有信号传递价值

## 亮点与洞察

- **科学仪器的天然多模态性**: CytoSense 每次测量自动产生配对数据——对比学习的理想场景。可推广到任何同时采集多信号的仪器。
- **k-NN 的标注效率**: 新站点只需少量标注——比重训分类器便宜得多。
- **弱模态也有价值**: 60% 的光学单独用不够，但通过对比学习它帮助图像编码器学到更鲁棒的特征。

## 局限性
- 物种数量有限；1D CNN 可换更强序列模型；无分割/检测能力

## 相关工作与启发
- **vs DINO 微调**: 不利用光学信号——对比学习额外获取了物理测量信息
- **启发**: 其他科学领域也有丰富多模态配对数据，对比学习可能是释放这些数据价值的通用范式
