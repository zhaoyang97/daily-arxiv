# Cross-modal Learning for Plankton Recognition

**日期**: 2026-03-17  
**arXiv**: [2603.16427](https://arxiv.org/abs/2603.16427)  
**领域**: 多模态/VLM  
**关键词**: 浮游生物识别, CLIP对比学习, 跨模态, 少样本, 光学剖面

## 一句话总结
将CLIP启发的对比学习应用于明场显微图像+光学剖面的跨模态浮游生物识别，用少量标注数据实现高精度分类和跨域泛化。

## 亮点
- 科学应用场景：海洋生态监测的自动化浮游生物分类
- 跨模态对比学习在小众领域的成功应用案例
