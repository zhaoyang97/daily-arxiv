# Visual Distraction Undermines Moral Reasoning in Vision-Language Models

**日期**: 2026-03-17  
**arXiv**: [2603.16445](https://arxiv.org/abs/2603.16445)  
**领域**: 机器人 / AI安全  
**关键词**: VLM道德推理, 视觉干扰, 安全对齐, 多模态伦理, 直觉-审思双系统

## 一句话总结
提出 Moral Dilemma Simulation (MDS) 基准，揭示视觉输入会绕过 VLM 的文本安全机制——视觉模态激活直觉式推理路径，覆盖纯文本场景中的审慎推理，导致道德判断退化。

## 亮点
- **"视觉绕过安全"的深刻发现**：文本安全对齐在多模态场景下失效——图片让模型"冲动"而非"理性"
- 认知科学的双系统理论在VLM中的验证：System 1（快/直觉/视觉驱动）vs System 2（慢/审慎/文本驱动）
- 对VLM安全部署有重大警示
