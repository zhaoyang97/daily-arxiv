# SemTok: Semantic One-Dimensional Tokenizer for Image Reconstruction and Generation

**日期**: 2026-03-17  
**arXiv**: [2603.16373](https://arxiv.org/abs/2603.16373)  
**领域**: 图像生成  
**关键词**: 1D图像token化, 语义对齐, 图像重建, 自回归生成, 紧凑表示

## 一句话总结
提出 SemTok，将2D图像压缩为语义对齐的1D token序列，通过语义对齐约束实现SOTA图像重建保真度，为下游自回归生成提供紧凑输入。

## 亮点
- **2D→1D降维**：打破了图像token必须保持2D空间结构的假设——1D序列更适合自回归生成
- **语义约束**：不仅重建像素还对齐语义——token携带高层含义
