# VIEW2SPACE: Studying Multi-View Visual Reasoning from Sparse Observations

**日期**: 2026-03-17  
**arXiv**: [2603.16506](https://arxiv.org/abs/2603.16506)  
**领域**: 3D视觉  
**关键词**: 多视图推理, 3D场景理解, 稀疏观测, 视觉证据链, 基准测试

## 一句话总结
提出 VIEW2SPACE 基准（2000个3D场景，3M QA对），评估从稀疏多视图进行3D推理的能力，提出基于视觉证据的 Grounded Chain-of-Thought 方法实现 +52% mIoU 提升并可迁移到真实世界。

## 亮点
- **3M QA对的大规模基准**：远超现有多视图推理数据集
- **+52% mIoU**：Grounded CoT 大幅提升空间推理——关键在于让模型展示推理依据的视觉证据
- **真实世界迁移**：合成场景训练可迁移到真实场景
