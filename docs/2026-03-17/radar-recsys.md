# RaDAR: Relation-aware Diffusion-Asymmetric Graph Contrastive Learning for Recommendation

**日期**: 2026-03-17  
**arXiv**: [2603.16800](https://arxiv.org/abs/2603.16800)  
**领域**: 图像生成 / 推荐系统  
**关键词**: 推荐系统, 扩散增强, 图对比学习, 关系感知, 稀疏数据

## 一句话总结
提出 RaDAR，结合扩散引导的图增强和非对称对比学习用于推荐系统，通过关系感知的边细化在噪声和稀疏条件下显著超越 SOTA。

## 亮点
- 扩散模型用于图结构增强（而非图像生成）——跨领域迁移
- 非对称对比学习处理用户-物品的天然非对称性
