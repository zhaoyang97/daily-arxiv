# MAPG: Multi-Agent Probabilistic Grounding for Vision-Language Navigation

**日期**: 2026-03-19 | **arXiv**: [2603.19166](https://arxiv.org/abs/2603.19166) | **领域**: 3D视觉 / 机器人
**关键词**: 视觉语言导航, 概率接地, 多智能体分解, 度量-语义推理

## 一句话总结
将度量-语义语言查询分解为多个组件，由VLM智能体分别接地并概率组合用于机器人导航，包含 MAPG-Bench 基准和真实机器人部署。

## 亮点: 概率组合确保语义不确定性被正确传播——比硬判断更稳健
