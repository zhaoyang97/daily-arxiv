# LVOmniBench: Pioneering Long Audio-Video Understanding Evaluation for Omnimodal LLMs

**日期**: 2026-03-19  
**arXiv**: [2603.19217](https://arxiv.org/abs/2603.19217)  
**代码**: [项目页面](https://kd-tao.github.io/LVOmniBench/)  
**领域**: 视频理解 / 多模态/VLM  
**关键词**: 长视频理解, 全模态评估, 音视频联合推理, OmniLLM, 基准测试

## 一句话总结
提出 LVOmniBench，首个专门评估全模态 LLM 在长时音视频（10-90分钟，共140小时）上的跨模态理解能力的基准，包含 275 视频+1014 人工标注 QA 对，发现开源模型准确率 <35%，Gemini 3 Pro 峰值仅 ~65%。

## 研究背景与动机

1. **领域现状**: 全模态 LLM（OmniLLM, 如 Qwen3 Audio-Video、Video-LLaMA2）同时处理音频+视频+文本。现有基准主要评估短片段（10秒-5分钟）。

2. **现有痛点**: 真实世界视频通常持续数十分钟（会议、电影、讲座），但没有基准评估 OmniLLM 在长时音视频上的表现。短视频基准无法反映长程记忆、时序定位和跨模态推理的挑战。

3. **核心 idea**: 构建 10-90 分钟长视频基准，要求音视频**联合推理**（仅看画面或仅听音频都无法回答），人工标注多难度等级。

## 基准亮点

- **规模**: 275 视频 (总 140 小时) + 1,014 QA 对
- **时长**: 10-90 分钟/视频——远超现有基准
- **严格手工标注**: 每个问题都需要音+视觉联合推理
- **多难度评估**: 分层评估长程记忆、时序定位、细粒度理解、多模态感知
- **多样来源**: 开放平台高质量视频，覆盖多种场景

## 实验关键数据

| 模型 | 准确率 |
|------|--------|
| Gemini 3 Pro | ~65% (最高) |
| 开源模型 (平均) | <35% |

**核心发现**：即使最强闭源模型也只有 65% 准确率——长时音视频理解仍是巨大挑战。开源模型与闭源差距约 30pp。

## 亮点与洞察

- **"音频不可缺"的设计**：每个问题都需要音频信息——不是视频问答加个背景音，而是真正的跨模态推理（如需要通过对话识别人物再计数）。
- **暴露了 OmniLLM 的长程能力短板**：当前模型在短视频上表现不错，但10分钟以上急剧退化——长程记忆和时序推理是系统性弱点。
- **难度分层**：区分记忆（在哪里提到了X）、定位（第几分钟发生了Y）、推理（为什么Z会导致W）等不同认知层次。

## 局限性
- 275 视频规模中等，某些类别可能样本不足
- 英语为主，多语言覆盖有限
- 评估格式为多项选择，无法衡量自由文本生成质量
