# PowerFlow: Unlocking the Dual Nature of LLMs via Principled Distribution Matching

**日期**: 2026-03-19  
**arXiv**: [2603.18363](https://arxiv.org/abs/2603.18363)  
**领域**: LLM推理  
**关键词**: 无监督强化学习, 分布匹配, GFlowNet, α幂分布, 推理与创造力

## 一句话总结
提出 PowerFlow，将无监督 LLM 微调形式化为 α-幂分布匹配问题——α>1 锐化分布增强推理、α<1 平化分布释放创造力——通过长度感知的 Trajectory-Balance 目标解决自回归长度偏差，在无外部监督下匹配甚至超越有监督的 GRPO。

## 研究背景与动机

1. **领域现状**: 无监督 RLIF（Reinforcement Learning from Internal Feedback）通过自确定性/一致性等内在奖励引导 LLM 自我进化，无需外部标注。

2. **现有痛点**: 启发式内在奖励缺乏理论目标——容易导致长度崩塌、过度自信、模式坍塌等退化行为。

3. **核心矛盾**: LLM 有"双重天性"——推理需要分布锐化（集中概率到正确路径），创造力需要分布平化（保持多样性）——但现有方法只能做一个方向。

4. **核心 idea**: 目标分布定义为基础分布的 α-幂：$p_\alpha(y|q) \propto p_{\text{base}}(y|q)^\alpha$。用 GFlowNet 做摊销变分采样器，长度感知 TB 目标消除自回归长度偏差。α 是唯一的可调参数——控制锐化/平化方向和强度。

## 方法亮点

- **理论驱动**：不是设计启发式奖励，而是直接匹配 α-幂分布——有明确的优化目标和收敛保证
- **长度感知 TB 目标**：将分区函数重参数化为逐 token 能量项，在长度归一化能量面上优化——完全消除自回归概率指数衰减的影响
- **双向能力激发**：α>1 增强推理 + α<1 释放创造力——用同一框架统一

## 实验关键数据

| 设定 | PowerFlow | GRPO (有监督) | 其他 RLIF |
|------|-----------|--------------|-----------|
| 推理 (α>1) | **匹配或超越** | 基线 | 弱于 |
| 创造力 (α<1) | **多样性+质量同时提升** | 过度锐化 | 未应用 |

- 在推理任务上无监督即可达到有监督 GRPO 的效果
- 在创造力任务上突破质量-多样性的 Pareto 边界

## 亮点与洞察

- **"分布锐化 = 推理提升"的理论基础**：将 RL post-training 的推理增益解释为隐式分布锐化，并提供可控的锐化/平化工具。
- **α 作为统一旋钮**：研究者可以用 α 精确控制"推理准确率 vs 创造多样性"——不再是二选一。
- **长度偏差是自回归生成的结构性问题**：不解决它，任何分布匹配都会退化——LA-TB 是必要的。
