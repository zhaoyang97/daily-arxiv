# Wasserstein Gradient Flows for Batch Bayesian Optimal Experimental Design

**日期**: 2026-03-12  
**arXiv**: [2603.12102](https://arxiv.org/abs/2603.12102)  
**代码**: 无  
**领域**: 多模态VLM / 贝叶斯优化  
**关键词**: Bayesian optimal design, Wasserstein gradient flow, batch design, information gain, particle methods

## 一句话总结
提出基于 Wasserstein 梯度流的批量贝叶斯最优实验设计方法——通过将批量 EIG 优化提升到概率测度空间，引入熵正则化（唯一 Gibbs 分布最小化器），推导 mean-field 和 i.i.d. 乘积族的梯度流公式，并用粒子时空离散化和双随机变体实现可扩展的近似求解。

## 研究背景与动机

1. **领域现状**: 贝叶斯最优实验设计（BOED）旨在选择最大化信息增益的实验配置，但批量设计（同时选择多个实验点）是组合优化难题。

2. **核心矛盾**: 批量设计空间随批大小指数增长，传统逐点贪心策略次优，且难以处理目标函数的多模态性。

3. **核心 idea**: 将批量设计问题重新表述为概率测度空间上的优化——用 Wasserstein 梯度流在连续空间中优化，自然避免组合爆炸。

## 方法详解

### 关键设计

1. **概率提升**: 将批量设计点的优化提升为概率测度上的优化
2. **熵正则化**: 正则化期望效用目标，得到唯一的 Gibbs 分布最小化器
3. **Wasserstein 梯度流**: 推导 mean-field 和 i.i.d. 乘积族的梯度流方程
4. **粒子离散化**: 时空离散化将连续流转化为可模拟的粒子算法
5. **双随机变体**: 结合交互粒子和蒙特卡洛 EIG 梯度估计

## 实验关键数据

| 特性 | 结果 |
|------|------|
| 多模态探索 | 能有效探索多模态优化景观 |
| 大批量可扩展 | 粒子算法支持大批量设计 |
| 长时行为 | 形式化刻画了梯度流的长时行为 |

## 亮点与洞察
- 将离散组合问题连续化为测度空间优化是优雅的数学处理
- Wasserstein 梯度流天然处理多模态——粒子不会塌缩到单一模式
- 理论贡献（Gibbs 分布刻画 + 长时行为分析）扎实

- 粒子方法在极高维度下的效率需要验证
