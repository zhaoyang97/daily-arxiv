# NEO: A Unified Language Model for Large-Scale Search, Recommendation, and Reasoning

**日期**: 2026-03-18 | **arXiv**: [2603.17533](https://arxiv.org/abs/2603.17533) | **领域**: 图像生成 / 推荐系统
**关键词**: 统一语言模型, 搜索推荐, 语义标识符, 目录对齐, 跨任务迁移

## 一句话总结
让 LLM 交织自然语言和语义标识符（SID）实现对 1000 万级目录的搜索/推荐/推理统一，单模型支持多媒体类型且跨任务正迁移。

## 亮点
- 1000万物品的搜索+推荐+推理统一到一个 LLM——无需额外工具
- 跨任务正迁移：搜索帮助推荐，推荐帮助搜索
