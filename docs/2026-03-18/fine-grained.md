# Fine-Grained Post-Training Quantization for LVLMs with Quantization-aware Integrated Gradients

**日期**: 2026-03-18 | **arXiv**: [2603.17809](https://arxiv.org/abs/2603.17809) | **领域**: 多模态/VLM / 模型压缩
**代码**: [GitHub](https://github.com/ucas-xiang/QIG)
**关键词**: 量化, LVLM, token级敏感度, 积分梯度, 后训练量化

## 一句话总结
借鉴可解释性中的公理化归因方法，提出量化感知积分梯度 (QIG) 将 LVLM 量化敏感度测量从模态级推进到 token 级，在 W3A16 下 LLaVA-onevision-7B 平均精度提升 1.60%，与全精度差距仅 1.33%。

## 研究背景与动机

1. **领域现状**: LVLM 后训练量化 (PTQ) 是降低部署成本的主流方法。现有 LVLM 量化方法（MBQ、QSLAW、Q-VLM）主要在模态级别衡量量化敏感度——给视觉 token 和文本 token 分别分配一个敏感度。

2. **现有痛点**: 观察到四个现象——大量激活离群值、层异质性、子层分歧、token 变异性。同一模态内不同 token 的量化敏感度差异巨大，模态级建模无法捕捉这些 token 级动态。且常用代理（注意力分数）与真实量化误差相关性弱。

3. **核心 idea**: 用积分梯度作为 token 级量化敏感度的度量。从量化参考输入到实际输入计算积分梯度路径，得到每个 token 对最终量化误差的归因分数，满足公理化归因的完备性和单调性。

## 方法详解

### 整体框架
校准阶段：对每个 Transformer block，计算 QIG token 级敏感度 → IQR 截断抑制极端值 → 用敏感度加权通道缩放因子的优化目标 → 标准 PTQ 量化。

### 关键设计

1. **量化感知积分梯度 (QIG)**:
   - 做什么：为每个 token 计算其对整体量化误差的归因分数
   - 核心思路：定义量化参考输入 $\bar{X}$（量化后激活），从 $\bar{X}$ 到实际输入 $X$ 沿直线路径积分梯度 $\text{QIG}_i = (X_i - \bar{X}_i) \cdot \int_0^1 \frac{\partial L}{\partial X_i}(\bar{X} + \alpha(X - \bar{X})) d\alpha$
   - 设计动机：满足完备性公理——所有 token 的 QIG 之和等于总量化误差，确保归因无遗漏

2. **IQR 截断稳定化**:
   - 做什么：抑制极端 token 敏感度值
   - 核心思路：用四分位距 (IQR) 方法截断过大/过小的 QIG 值，防止少数 token 主导整个优化
   - 设计动机：量化场景下离群值普遍存在，不截断会导致优化不稳定

3. **敏感度加权校准目标**:
   - 做什么：将 token 级敏感度整合到通道缩放因子优化中
   - 核心思路：在标准 CWE 目标中用 QIG 分数对不同 token 的重建误差加权，高敏感 token 获得更大权重

## 亮点与洞察
- **可解释性→压缩的跨领域迁移**：积分梯度原本用于模型解释，本文首次将其用于指导量化决策
- **Token 级异质性被低估**：同模态内 token 敏感度差异可达数量级，模态级建模损失大量信息
- **训练无关且延迟可忽略**：仅在校准阶段计算 QIG，推理时无额外开销

## 局限性 / 可改进方向
- 积分梯度计算需多次前向-反向传播（10-20 步），校准时间增加
- 仅在 W3A16 和 W4A8 验证，更极端的 2-bit 场景效果未知
- 假设量化参考到实际输入的直线路径是最优积分路径，可能不是
