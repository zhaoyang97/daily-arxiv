# GeCO: Time Unconditional Flow Matching for Robotic Control

**日期**: 2026-03-18 | **arXiv**: [2603.17834](https://arxiv.org/abs/2603.17834) | **领域**: 图像生成 / 机器人
**关键词**: Flow Matching, 机器人控制, 无训练OOD检测, 自适应推理

## 一句话总结
将动作生成从固定积分转变为自适应迭代优化（使用静止速度场），提供无训练的 OOD 检测和基于任务复杂度的计算预算分配。

## 亮点
- 静止速度场=无需时间条件：简化 FM 架构
- 自适应推理——简单动作用少步，复杂动作用多步（类似 ProbeFlow 的思路但方法不同）
