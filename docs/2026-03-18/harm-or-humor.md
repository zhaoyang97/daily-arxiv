# Harm or Humor: A Multimodal, Multilingual Benchmark for Overt and Covert Harmful Humor

**日期**: 2026-03-18  
**arXiv**: [2603.17759](https://arxiv.org/abs/2603.17759)  
**领域**: 多模态/VLM / AI安全  
**关键词**: 有害幽默检测, 多模态安全, 多语言, 暗示性内容, 文化推理

## 一句话总结
提出多模态多语言有害幽默检测基准（3000文本+6000图像+1200视频，英语/阿拉伯语），将有害幽默细分为显式和隐式两类，发现闭源模型显著优于开源，阿拉伯语表现远差于英语，凸显文化感知安全对齐的迫切需求。

## 研究背景与动机

1. **领域现状**: AI 安全评估聚焦于直接有害内容（仇恨言论、暴力），但忽略了一种更微妙的形式——有害幽默（黑色幽默、攻击性玩笑），尤其是需要文化背景和隐含推理才能识别的隐式有害幽默。

2. **现有痛点**: 现有毒性基准通常是静态的纯文本数据集，无法捕捉多模态（图+文+视频）场景下的微妙有害信号。跨语言表现差距大但缺乏系统评估。

3. **核心 idea**: 构建涵盖文本、图像、视频的多模态数据集，区分安全/显式有害/隐式有害三类，支持英语+阿拉伯语，评估 SOTA 模型。

## 基准亮点

- **规模**: 3,000 文本 + 6,000 图像 + 1,200 视频
- **语言**: 英语 + 阿拉伯语 + 语言无关（通用）
- **三级标注**: Safe → Harmful → Explicit / Implicit
- **人工标注**: 严格标注指南确保质量
- **关键区分**: "隐式 Implicit"有害幽默需要深层推理才能识别——这是现有模型的最大弱点

## 实验关键数据

| 发现 | 说明 |
|------|------|
| 闭源 >> 开源 | 有害幽默检测差距显著 |
| 英语 >> 阿拉伯语 | 两类模型都存在跨语言差距 |
| 隐式 >> 显式（难度） | 模型对隐式有害幽默检测更差 |
| 多模态增加难度 | 图像/视频中的隐含线索更难捕捉 |

## 亮点与洞察

- **"隐式有害幽默"是安全盲区**：直接的仇恨言论容易检测，但包装成笑话的歧视更隐蔽——现有安全过滤器很可能漏掉这类内容。
- **阿拉伯语安全对齐严重不足**：凸显了AI安全研究的英语中心偏见——其他语言和文化的安全防护远远落后。
- **多模态是必要条件**：很多有害幽默只有在图文结合时才能理解（如讽刺性配图），纯文本评估不够。
