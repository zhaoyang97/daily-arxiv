# LED: A Benchmark for Evaluating Layout Error Detection in Document Analysis

**日期**: 2026-03-18 | **arXiv**: [2603.17265](https://arxiv.org/abs/2603.17265) | **领域**: 多模态/VLM
**关键词**: 文档布局, 错误检测, 结构理解, LLM评估

## 一句话总结
提出 LED 基准系统性诊断文档布局预测中的 8 种结构错误类型，实现对 LLM 和多模态模型结构理解能力的细粒度评估。

## 亮点
- 8 种错误类型的系统分类——填补文档布局评估的细粒度空白
- 标准化评估框架可跨模型对比
