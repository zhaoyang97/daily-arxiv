# Learning Transferable Temporal Primitives for Video Reasoning via Synthetic Videos

**日期**: 2026-03-18 | **arXiv**: [2603.17693](https://arxiv.org/abs/2603.17693) | **领域**: 视频理解
**关键词**: 时序推理, 合成视频, 时序原语, CoT训练, RL

## 一句话总结
用合成几何视频构造 7.7K CoT + 7K RL 样本教授方向/速度/状态追踪等时序原语，7.7K 合成样本在 15 个时序推理基准上超越 165K 真实样本。

## 亮点
- **7.7K 合成 > 165K 真实**：时序推理的核心是抽象原语而非视觉复杂度——合成数据完美匹配
- 对VLM时序推理能力的根本洞察
