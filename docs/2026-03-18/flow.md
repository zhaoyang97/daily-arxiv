# Flow Matching Policy with Entropy Regularization (FMER)

**日期**: 2026-03-18 | **arXiv**: [2603.17685](https://arxiv.org/abs/2603.17685) | **领域**: 图像生成 / 强化学习
**关键词**: Flow Matching, 熵正则化, ODE策略, 探索增强, RL

## 一句话总结
基于 ODE 的 RL 框架，将 Flow Matching 与可解析的熵正则化结合增强探索，训练速度比 QVPO 快 7×，比高效扩散变体快 10-15%。

## 亮点
- FM + 可解析熵——ODE 比 SDE 更高效但通常缺少随机性（探索），熵正则化弥补这一gap
- 7× 训练加速
