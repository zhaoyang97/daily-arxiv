# SegFly: A 2D-3D-2D Paradigm for Aerial RGB-Thermal Semantic Segmentation

**日期**: 2026-03-18 | **arXiv**: [2603.17920](https://arxiv.org/abs/2603.17920) | **领域**: 3D视觉
**关键词**: 航拍分割, RGB-热, 自动标注, 多视图几何, 标签传播

## 一句话总结
用多视图几何从 <3% 手动标注的 RGB 图像自动传播标签到 RGB+热成像，构建 20K+ RGB + 15K+ RGB-T 对的基准，标注准确率达 91%/88%。

## 亮点
- **<3% 手动标注→91% 准确率**：多视图几何做自动标注极其高效
- RGB-热航拍分割新基准——填补数据空白
