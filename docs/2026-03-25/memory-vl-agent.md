# Memory-Augmented VL Agents for Persistent Object Captioning

**日期**: 2026-03-25  
**arXiv**: [2603.24257](https://arxiv.org/abs/2603.24257)  
**代码**: 无  
**领域**: 多模态VLM / Agent  
**关键词**: VLM agent, memory, consistent captioning, multi-view, embodied

## 一句话总结

提出记忆增强的视觉语言 Agent 框架，通过维护持久化物体级语义记忆在线聚合多视角观测，解决 VLM 对同一物体在不同视角下产生不一致描述的问题，无需离线多阶段处理即可保持跨时间和视角的描述一致性。

## 研究背景与动机

1. **领域现状**：VLM 在单帧 captioning 上表现出色，但对同一物体从不同角度观察时常产生矛盾描述。这阻碍了具身 agent 构建一致的语义表示。
2. **现有痛点**：(a) 离线多视角聚合需收集所有视角后处理，不适合在线；(b) 多阶段方法流水线复杂
3. **核心矛盾**：实时性 vs 一致性——在线 agent 需即时描述但要保一致
4. **核心 idea**：**物体级持久化记忆——在线维护每个物体的语义表示，新观测增量更新，描述基于累积记忆生成**

## 方法详解

### 关键设计
1. **物体级语义记忆**：为每个识别到的物体维护持久化语义向量，跨帧更新
2. **在线记忆更新**：新视角观测通过注意力机制融入已有记忆，增量更新而非覆盖
3. **记忆引导描述生成**：captioning 基于累积记忆而非单帧特征

## 实验关键数据

| 方法 | 一致性↑ | BLEU | 在线性 |
|------|--------|------|--------|
| LLaVA (单帧) | 低 | 好 | 在线 |
| 离线多视角聚合 | 高 | 好 | 离线 |
| **Memory-VL Agent** | **高** | **好** | **在线** |

| 记忆粒度 | 一致性 | 说明 |
|---------|--------|------|
| 全局记忆 | 中 | 粗粒度 |
| 帧级记忆 | 中偏上 | 仍有冗余 |
| **物体级记忆** | **最优** | 实体粒度最合适 |

### 关键发现
- 物体级记忆优于帧级和全局级——实体粒度匹配了"一致性"的语义单位
- 记忆增强在一致性上接近离线方法，同时保持在线实时性

## 亮点与洞察
- **在线一致性的实际需求**：机器人导航中需实时且一致的场景描述
- **记忆粒度选择的重要性**：物体级 > 帧级 > 全局级

## 局限性 / 可改进方向
- 物体追踪失败时记忆断裂
- 长时间运行的记忆增长问题未解决
