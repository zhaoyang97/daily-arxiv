# OmniWeaving: Towards Unified Video Generation with Free-form Composition

**日期**: 2026-03-25  
**arXiv**: [2603.24458](https://arxiv.org/abs/2603.24458)  
**代码**: 无  
**领域**: 视频生成 / 多模态  
**关键词**: unified video generation, free-form composition, open-source, video editing, video synthesis

## 一句话总结

OmniWeaving 是一个开源的统一视频生成框架，将文本生成、图像驱动、视频编辑、风格迁移等多种视频生成任务整合到一个模型中，支持自由形式的条件组合，缩小了与闭源系统（Seedance-2.0）的差距。

## 研究背景与动机

1. **领域现状**：闭源视频生成系统（Seedance-2.0、Sora）已实现多功能统一，但开源替代方案严重碎片化——文生视频、图生视频、视频编辑各用不同模型。
2. **现有痛点**：(a) 学术模型功能单一，不能一个模型做多任务；(b) 少数尝试统一的开源方案难以无缝融合不同条件；(c) 条件组合（如"参考图 + 文本 + 运动引导"同时输入）支持差
3. **核心矛盾**：统一意味着模型要理解和组合异质条件信号，但各条件的表示空间不同
4. **切入角度**：设计自由形式的条件编码方案，将所有条件类型统一到同一表示空间
5. **核心 idea**：**条件表示统一化 + 自由组合——将文本、图像、视频、运动等异质条件映射到统一 token 空间，支持任意组合输入**

## 方法详解

### 整体框架

基于视频扩散模型，OmniWeaving 将所有条件（文本描述、参考图像、运动轨迹、风格参考等）通过各自的编码器映射为条件 token，拼接后输入去噪网络的 cross-attention。训练时随机组合不同条件，推理时用户自由选择。

### 关键设计

1. **统一条件编码**：每种条件通过适配器映射到同维 token 空间
2. **混合训练策略**：随机采样条件子集训练，学习任意组合
3. **质量-多样性平衡**：分阶段训练——先单条件后多条件组合

## 实验关键数据

| 方法 | 文生视频 FVD↓ | 图生视频质量 | 视频编辑 | 统一模型 |
|------|-------------|-----------|---------|---------|
| CogVideoX | 好 | 无 | 无 | 否 |
| I2VGen-XL | 无 | 好 | 无 | 否 |
| **OmniWeaving** | **竞争力** | **好** | **好** | **是** |

### 关键发现
- 统一模型在单任务上不比专用模型差——多任务训练有正向迁移
- 自由组合条件的能力是关键差异化——用户可以同时指定"参考人脸+运动轨迹+风格"
- 与 Seedance-2.0 仍有差距但大幅缩小了开源-闭源鸿沟

## 亮点与洞察
- **开源统一方案的重要性**：填补了开源视频生成的"功能碎片化"问题
- **自由组合条件**：不是固定几种模式，而是任意组合——更接近实际创作需求

## 局限性 / 可改进方向
- 条件数多时质量可能下降
- 长视频生成一致性待提升
