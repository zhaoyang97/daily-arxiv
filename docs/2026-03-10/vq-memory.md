# VQ-Memory: Beyond Short-Horizon for Robust Long-Horizon Manipulation

**日期**: 2026-03-10  
**arXiv**: [2603.09513](https://arxiv.org/abs/2603.09513)  
**代码**: 有  
**领域**: 机器人 / 强化学习  
**关键词**: long-horizon manipulation, VQ-VAE, memory, non-Markovian, simulation

## 一句话总结
提出 RuleSafe 基准（LLM 生成的多阶段解锁规则，具有非马尔可夫性质）和 VQ-Memory 模块——通过 VQ-VAE 将关节状态历史编码为离散语义 token 再做 K-means 聚类，实现 ~20 倍压缩的紧凑记忆表示，使 VLA 模型能有效处理长视野多阶段操纵任务。

## 研究背景与动机

1. **领域现状**: 机器人仿真基准多聚焦短视野任务，实际场景需要多步骤、非马尔可夫的长期操纵。
2. **现有痛点**: (i) 原始关节状态作为记忆容易过拟合低层噪声；(ii) 视觉历史计算开销大；(iii) 缺乏标准化的非马尔可夫操纵基准。
3. **核心 idea**: 用 VQ-VAE 的离散码本学习紧凑的语义记忆表示，过滤低层噪声保留高层任务信息。

## 方法详解

### VQ-Memory 核心设计
1. **VQ-VAE 编码**: 50 帧窗口的关节状态 → 256 维码本
2. **K-means 聚类**: 从 256 码字下采样到 4 簇
3. **紧凑表示**: ~20 倍压缩比
4. **模型无关**: 适配 DP3/RDT/CogACT/π0 等多种策略

### RuleSafe 基准
- 20 条 LLM 生成的解锁规则（密钥锁、密码锁、逻辑锁等）
- 结合部件阶段（关节配置）和任务阶段（进度追踪）
- 非马尔可夫性质——必须记住过去的操作

## 实验关键数据
- Rule_020: VQ-Memory 45% 成功率 vs 无记忆 0%
- 多任务平均成功率 56.3% vs 无记忆 25%
- 4 簇设置最优，过多簇导致过度细化
- 跨 4 种策略架构一致有效

## 亮点与洞察
- **VQ 量化** 优雅解决连续记忆的噪声问题——离散化天然过滤低频抖动
- **聚类实现语义发现**——不同簇对应不同任务阶段
- **模块化设计**便于即插即用

## 局限性 / 可改进方向
- 聚类簇数需手动选择，缺乏自适应机制
- 低层操纵精度仍为瓶颈（56% vs 理想 100%）
