# LLM-MRD: LLM-Guided Multi-View Reasoning Distillation for Fake News Detection

**日期**: 2026-03-10  
**arXiv**: [2603.19293](https://arxiv.org/abs/2603.19293)  
**代码**: 有  
**领域**: 多模态/VLM / 假新闻检测  
**关键词**: fake news detection, knowledge distillation, multi-view reasoning, LLM, calibration

## 一句话总结
提出 LLM-MRD，一个教师-学生框架：LLM 教师从文本/视觉/跨模态三个视角生成深度推理链作为监督信号，通过校准蒸馏（Calibration Distillation）将推理知识高效转移到轻量学生模型，在三个多模态假新闻检测基准上平均提升 ACC 5.19%、F1-Fake 6.33%。

## 研究背景与动机

1. **领域现状**: 社交媒体假新闻泛滥，需要自动检测。现有方法分为单模态、多模态融合和 LLM 增强三类。

2. **现有痛点**: (i) 缺乏全面的多视角判断和融合——大多方法只做浅层特征融合；(ii) LLM 推理虽强但推理成本过高，无法直接部署。

3. **核心 idea**: 用 LLM 作为专家教师进行多视角深度推理，然后通过校准蒸馏把推理知识转移到高效学生模型——从"特征融合"转向"层次化推理知识迁移"。

## 方法详解

### 学生多视角推理
- **文本视角**: BERT 编码 + 自注意力 → $f_{\text{text}}$
- **图像视角**: MAE 编码 + 自注意力 → $f_{\text{image}}$
- **跨模态视角**: CLIP 对齐编码 + 协注意力 → $f_{\text{cross}}$

### 教师多视角推理
- Qwen2.5-VL 作为 MLLM 教师
- 分别从文本/图像/跨模态三个角度生成推理链
- 用 Sentence Transformer 编码为稠密向量

### 校准蒸馏（核心创新）
- 不是简单投影对齐，而是让学生预测"校正向量" $d_v^{\text{pred}}$
- 输入是三个视角特征的拼接（全局上下文），输出视角特定的校正
- 残差方式：$\hat{f}_v = f_v + d_v^{\text{pred}}$——学生自我纠正
- 损失：KL 散度（对齐教师）+ 交叉熵（深度监督）

### 多视角融合
- 校准后特征池化为全局查询
- 跨注意力动态挖掘各模态中最相关的证据

## 实验关键数据

### 主实验

| 方法 | Weibo Acc | Weibo F1 | Weibo21 Acc | GossipCop Acc |
|------|-----------|----------|-------------|---------------|
| MIMoE-FND | 0.928 | 0.928 | 0.956 | 0.895 |
| KEN | 0.935 | 0.935 | 0.935 | 0.881 |
| FactAgent | 0.927 | 0.943 | 0.932 | 0.860 |
| **LLM-MRD** | **0.953** | **0.954** | **0.959** | **0.902** |

### 关键发现
- 在 Weibo 上 ACC 提升 1.93%、AUC 提升 1.54%
- 跨三个基准（中/英文）一致领先，证明方法的跨域泛化
- 校准蒸馏优于直接投影蒸馏——残差式自我纠正有效
- 学生推理时间远低于直接用 LLM

## 亮点与洞察
- **校准蒸馏的残差设计** 很有洞察——不是让学生模仿教师输出，而是让学生预测"自己和教师的差距"，从而自我纠正
- **三视角对齐** 确保每个模态的知识都被充分利用，而非只融合最终特征
- 教师推理链转化为向量监督的方式 实用且高效——避免了让学生模型生成文本推理的开销

## 局限性 / 可改进方向
- 教师推理链的质量依赖 Qwen2.5-VL，可能存在幻觉
- 三个视角的权重 λ 是统一的，未考虑不同样本的模态重要性差异
- 仅在假新闻检测上验证，框架可推广到其他多模态判别任务
