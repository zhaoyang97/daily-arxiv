# Evolving Prompt Adaptation for Vision-Language Models

**日期**: 2026-03-10  
**arXiv**: [2603.09493](https://arxiv.org/abs/2603.09493)  
**代码**: 有  
**领域**: 多模态/VLM / 提示学习  
**关键词**: Vision-Language Models, prompt adaptation, distribution shift, test-time adaptation, evolutionary

## 一句话总结
提出动态演化提示自适应方法，使视觉语言模型能够在测试时逐步调整文本提示以应对分布漂移，通过演化算法在提示空间中搜索最优适配，无需目标域数据标签即可实现高效模型适应。

## 研究背景与动机

1. **领域现状**: 视觉语言模型（VLM）如 CLIP 在跨域转移时存在显著的性能下降。
2. **现有痛点**: (i) 现有提示调优方法（CoOp/CoCoOp）需要少量标注数据；(ii) 测试时适应方法依赖固定的提示策略；(iii) 手动提示工程费时且效果不稳定。
3. **核心 idea**: 将提示优化视为演化搜索问题，利用 CLIP 的零样本评估能力作为适应度函数，通过遗传算法在测试时动态搜索最优提示。

## 方法详解

### 演化提示自适应框架
- 将提示编码为可进化的符号序列
- 利用图像-文本相似度作为适应度评分函数
- 采用遗传算法进行种群迭代——选择、交叉、变异
- 变异算子支持词替代、插入、删除等操作

### 关键设计
- 每代保留高适应度提示并变异产生后代
- 不修改模型参数，完全在提示空间操作
- 通用框架适用于不同 VLM 架构

## 实验关键数据
- 在 Oxford Pets、Food101 等多个数据集上评估
- 相比固定提示提升 5-15% 的准确率
- 与 DPT、CoOP 等方法相比性能更优
- 无需目标域标签的完全无监督适应

## 亮点与洞察
- **演化算法用于提示搜索** 天然支持多样性探索，避免单一梯度方向
- 无需训练或微调，纯推理时优化
- 框架具有通用性，可应用于不同 VLM

## 局限性 / 可改进方向
- 演化过程计算开销较高（约推断时间的 5 倍）
- 提示空间搜索可能陷入局部最优
- 对初始提示种群敏感
