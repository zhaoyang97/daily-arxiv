# Daily arXiv

每日自动追踪 arXiv 上 AI / LLM / NLP / CV 领域新发布的论文，筛选最值得关注的论文并生成深度论文解读。

## 在线浏览
[zhaoyang97.github.io/daily-arxiv](https://zhaoyang97.github.io/daily-arxiv/)

## 特色

- 每天从 arXiv 获取前一天最新论文,  打分筛选出最值得关注的 ~20 篇，生成深度阅读笔记（100+ 行）
- 每日速览页：领域分布 + 全部论文一览表
- 自动部署到 GitHub Pages，推送即发布

## 追踪类别

| 类别 | 全称 | 说明 |
|------|------|------|
| `cs.CV` | Computer Vision | CV 核心 |
| `cs.CL` | Computation and Language | NLP/LLM 核心 |
| `cs.AI` | Artificial Intelligence | AI 通用 |
| `cs.LG` | Machine Learning | ML 核心 |
| `cs.MM` | Multimedia | 多模态 |
| `cs.IR` | Information Retrieval | 检索/RAG |
| `cs.RO` | Robotics | 具身智能 |
